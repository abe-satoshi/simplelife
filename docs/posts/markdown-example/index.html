Markdownの書き方：備忘録

ブログでも仕事でも、Markdownが使えると便利。自分の備忘録を兼ねて。


出典：Wikimedia Commons

見出し


「#」の後に空白でタイトル、「#」の数で異なる

# タイトル h1
## タイトル h2
### タイトル h3
#### タイトル h4

SEOの観点から、ブログのタイトルはh1、記事は、h2、Markdownでは「##」から使うのが一般的。
